# 📝 Notion-like Notes App (Web Exercise 1)

## 📌 Project Description

This project is a simplified Notion-like note-taking application built as part of a university web programming exercise.

The goal is to practice building a modern web app with **HTML, CSS, and JavaScript**, while implementing features similar to Notion such as notes, folders, search, markdown support, and local data persistence.

---

## ✨ Features

### 🗂 Notes & Folders
- Create, edit, and delete notes
- Create folders (single-level hierarchy only)
- Move notes between folders
- Pin important notes

### 📝 Note Editor
- Title and content support
- Basic Markdown support:
  - Headings (#, ##)
  - Bold (**text**)
  - Italic (*text*)
  - Inline code (`code`)
  - Code blocks (``` ```)

### 📂 Sidebar
- Always visible navigation sidebar
- List of folders and notes
- Pinned notes section
- Search bar for filtering notes
- Buttons for creating notes and folders

### 🔍 Search & Sorting
- Search notes by title or content
- Sort notes alphabetically or by last modified date

### 💾 Data Persistence
- All data is saved automatically using `localStorage`
- Data is restored after page reload

### 📤 Export / Import
- Export all data as JSON file
- Import JSON file to restore notes and folders

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- LocalStorage API
- JSON handling

---

## 🎯 Learning Goals

- DOM manipulation
- State management in vanilla JS
- Working with localStorage
- File import/export using JSON
- Building a UI similar to Notion
- Handling user interactions (CRUD operations)
